# GitHub Copilot Instructions for EED Project

This file provides context for GitHub Copilot when working on this project.

## Project Overview
Energetic Epicenter Detector (EED) - Computer vision engine for detecting energetic event origins in video using optical flow and fluid dynamics.

## Code Style
- Follow PEP 8
- Use type hints where appropriate
- Comprehensive docstrings with Args/Returns/Raises
- Maximum line length: 100 characters

## Key Dependencies
- OpenCV (cv2) - Video processing and optical flow
- NumPy - Numerical computations
- SciPy - Scientific computing (ndimage, signal)
- Matplotlib - Visualization

## Architecture
- `energetic_detector.py` - Core physics engine
- `epicenter_tool.py` - AI integration wrapper
- Tests in `tests/` directory using unittest

## Testing
- All new features need unit tests
- Run: `python -m unittest discover tests`
- Tests should be in `tests/` directory

## Common Patterns
- Use EnergeticEpicenterDetector class for detection
- Use EpicenterTool for AI integration
- JSON output for programmatic use
- Visualization for human review

## Important Notes
- Videos use OpenCV VideoCapture
- Optical flow via cv2.calcOpticalFlowFarneback
- Temporal weighting prioritizes early frames
- Results include x, y coordinates and confidence scores
